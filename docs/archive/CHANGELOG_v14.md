# SCM 대시보드 v14 변경 내역 (2026.07.14)

팀 공유용 — 변경 1건당 1줄 요약입니다. 상세 사용법은 `docs/SCM_DASHBOARD_사용자가이드_v14.html` 참고.

v14는 파트장님이 원격에 올린 v13 작업과, 같은 날 별도 세션에서 진행하던 검색 필터 수정 작업을 병합한 버전입니다. v13 변경 내역은 `docs/archive/CHANGELOG_v13.md`에 그대로 남아있습니다.

## 검색 필터
- 발주 TASK 상세 모달(발주현황Top/미발주현황/DB원가비교) — 기간 필터만 걸리고 검색어가 무시되던 문제 수정
- 매입추이 월클릭 상세 모달 — 클릭한 월의 전체 데이터가 뜨던 문제 수정, 검색 결과만 표시
- 이슈 상세 모달의 품질/수량/운영 서브탭 — "전체" 탭에서만 검색이 유지되고 서브탭 전환 시 풀리던 문제 수정
- 이슈 유형 분포 도넛, 월별 이슈 추이 차트 — 대시보드 화면 단계부터 검색 필터가 전혀 적용되지 않던 문제 수정
- 고객인지이슈(ci) 검색 매칭 신규 — ci.csv에 협력사 컬럼이 없어 협력사 검색이 되지 않던 문제를, 프로젝트명을 발주_RAW의 project로 1차 조회한 뒤 해당 프로젝트 내 산출물 중 관련제품과 가장 가까운 행의 수주처로 연결해 해결 (프로젝트 범위로 한정 — 전체 매칭 대비 과다매칭 방지)

## 원본 데이터
- issue.csv에 `입고물품` 컬럼 추가 — 이슈 목록·검색에 표시되는 제품명을 이 컬럼 우선으로 변경(기존 `item (통합) (from order)`는 뒤섞인 표기가 많았음). 제품×협력사 이슈율 매트릭스의 order.csv 조인 필드는 정확 일치가 필요해 기존 필드를 그대로 유지
- ci.csv 인코딩을 CP949 → UTF-8(BOM)로 수정 — 컬럼 인식 오류("등록일자"/"이슈내용" 없음) 해결

## 신규 기능
- 현재 화면(검색·기간 필터 적용) 표를 CSV로 다운로드하는 기능 추가 — 상단바 다운로드 버튼, 화면에 보이는 그대로 내려받음

## 파트장님 작업 (v13, 병합됨)
- 발주 진행현황 탭 신규, 기간 통합 선택(SEL_RANGE), 재고생산 TASK·매입 별도 KPI 카드, 이슈 상세 3단 구성, 고객인지이슈 대응내용·실패비용 수기입력, GitHub 자동 커밋 연동 등 — 전체 목록은 `docs/archive/CHANGELOG_v13.md` 참고

## 운영·안정성
- `deploy_v13.ps1` → `deploy_v14.ps1` 갱신(버전 넘버링만 변경, 팀장님 PC 배포 경로는 동일)
- 여러 담당자가 동시에 수정한 두 브랜치(원격 v13 force-push / 로컬 세션 작업)를 `git merge`로 병합 — 겹치는 함수는 양쪽 로직을 결합(v13의 기간·3단구성 로직 + 세션의 검색 필터 재적용)하는 방식으로 수동 해소

## 다음부터
- 대시보드를 변경하면 변경 내용을 이 파일(다음 버전은 `docs/CHANGELOG_v{N}.md`)에 기록해 공유
- 수정 반영 후에는 버전 관리(스냅샷·배포스크립트·문서 갱신)를 함께 진행

# 추가 변경 내역 (2026.07.15)

위 v14 병합과는 별도로 같은 날 로컬에서 진행된 작업. 상세 사용법은 `docs/SCM_DASHBOARD_사용자가이드_v14.html`(§7, §13), 로직은 `docs/SCM_DASHBOARD_로직설명_v14.html`(§3-13, §3-20) 참고.

## 월간 공지사항 카드 신규
- 종합 현황 상단에 졸업 제품·출시 제품(`goods_master.csv` 졸업일/출시일)·품절 제품(`stockout_list` 판매상태+변경일)을 선택 월 기준 자동 집계
- 신규 CSV 슬롯 `goods_master` 추가(Airtable Sincerely DB "1. goods" 뷰)
- 협력사 신규/거래종료를 `sup.csv`의 이번 달 명단과 `CSV_BANK/sup_YYYY_MM.csv`(직전 달 스냅샷)를 이름 기준으로 비교해 자동 집계 — `archive_csv.ps1`이 매달 교체 직전 `sup.csv`를 스냅샷으로 저장하도록 확장
- 스냅샷 diff로 못 잡는 예외(상태변경일 필드가 없어서, 또는 이번 달 변경일이 아니라서)는 `MANUAL_VENDOR_TERMINATIONS`/`MANUAL_STOCKOUT` 배열로 특정 월에만 수동 반영 (거래종료 6곳: 베러웨이시스템즈·포텍·유경·파워·아이큐아이·모아산업 — 6월 귀속(26.2Q 마지막 달) / 강천FNT는 아직 확정 전이라 제외)
- 품절 제품 표기를 "PT번호-제품명"만 남기도록 간소화

## 협력사 현황 — 목록 드릴다운 전환
- "협력사 현황" 탭의 협력사 목록에서 이름 클릭 시 모달 팝업 대신 테이블 내 인라인 드릴다운으로 펼침(여러 곳 동시 확장 가능)
- 협력사명 검색 결과가 1곳으로 좁혀지면 자동으로 펼침
- 모달(다른 탭에서 호출)과 드릴다운이 `buildSupplierDetail()` 공용 빌더를 공유하도록 리팩터링

## 협력사 상세 콘텐츠 확장
- 매입추이 그래프 위에 월별 금액 + 전월 대비(MoM) 증감률 스트립 추가
- "발주 제품 전체"를 발주 횟수 내림차순 전체 목록으로 확장(기존 Top3 대체), 발주 이력 없는 제품은 제외, 발주 수량 병기
- "발주내역" 테이블 신규 — 개별 발주 건(발주번호·산출물·수량·공급가액·일자)
- 월별 필터 드롭다운 신규 — 선택한 달의 매입액 요약·발주내역·이슈 상세를 함께 필터링

## 화면 가독성
- 월간 공지사항 카드, 발주 진행현황 상세 표의 글씨 크기 확대
