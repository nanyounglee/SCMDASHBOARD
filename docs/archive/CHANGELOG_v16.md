# SCM 대시보드 v16 변경 내역 (2026.07.16)

팀 공유용 — 변경 1건당 1줄 요약입니다. 상세 사용법은 `docs/SCM_DASHBOARD_사용자가이드_v16.html`, 구현 로직은 `docs/SCM_DASHBOARD_로직설명_v16.html` 참고. v15/v15.x(기간선택 통합·월간 공지사항·이슈현황 탭 기간 필터/집계 기준일 수정) 내역은 `docs/archive/CHANGELOG_v15.md` 참고.

v16은 **협력사 현황 화면에 PDF 내보내기 2종을 추가**하고, 기존 **리포트/내보내기 버튼 2곳의 집계 버그를 KPI 카드 기준으로 바로잡은** 버전입니다. 협력사 드릴다운을 펼친 상태에서 화면 그대로를 PDF로 저장하거나, 전년(2025) 실적과 비교한 연간 분석 리포트를 자동 생성할 수 있습니다.

## 협력사 상세 PDF 저장 (신규)
- 협력사 현황에서 협력사를 펼치면(드릴다운) 우측에 **`📄 PDF 저장`** 버튼이 생깁니다 — 지금 보고 있는 협력사 상세 화면(매입추이·발주내역·이슈 등)을 그대로 A4 PDF로 저장합니다
- 화면에서 스크롤로 잘려 보이던 표도 PDF에는 전부 펼쳐서 출력되고, 월별 필터·버튼 같은 조작용 요소는 빠집니다 (선택한 월은 PDF 상단 머리말에 표기)
- 매입추이 차트는 캡처가 불안정한 canvas 대신 이미지(PNG)로 바꿔 넣어 안정적으로 나옵니다
- 파일명: `협력사상세_협력사명_YYYY-MM-DD.pdf`

## 협력사 분석 리포트 (신규)
- 같은 드릴다운에 **`📊 분석 리포트`** 버튼 추가 — 전년(2025) 대비 올해 실적을 비교·전망하는 A4 2페이지 분석 리포트를 자동 생성합니다
- 구성: ① 연간 요약·YoY(전년 동기/연간 예상) ② 월별 발주 추이 추세선(실적 실선 + 예상 점선) + 월별 표 ③ 발주 상위 제품 Top 5(전년/올해) ④ 동일 제품×유형 이슈 재발 케이스 ⑤ 자동 생성 핵심 요약 2줄
- **예상치 산식**: 클릭한 달의 전월까지를 "완결 실적"으로 보고, 그 구간의 YoY 성장률(TASK·수량·금액 각각)을 전년 같은 달 실적에 곱해 잔여 월을 채웁니다 — 계절성은 전년 패턴을 그대로 반영. 리포트에서 파란 숫자가 예상치입니다
- 전년(2025) 원본 CSV(`SCM_발주_RAW(2025).csv` / `SCM_이슈_RAW(2025).csv`)를 최초 1회 읽어 계산합니다. 전년 데이터가 없거나 1월에 열면 YoY·예상을 생략하고 올해 실적만 표기합니다
- 파일명: `협력사분석리포트_협력사명_YYYY-MM.pdf`

## 리포트/내보내기 버그 수정
- **주간 업무 보고(리포트 생성)의 발주 현황 수치를 KPI 카드와 일치**시켰습니다 — ① 긴급 발주 건수에서 서울디지털인쇄협동조합을 제외(카드와 동일 기준, 그동안 카드보다 과다집계) ② 월 매입금액에 재고생산분 포함(v9 카드 기준, 그동안 `!isStock` 필터로 과소집계). 재고생산 TASK 건수도 별도 병기하도록 표기 개선
- **`KPI 요약 CSV 내보내기` 버튼이 실제 KPI 요약을 내보내도록 수정**했습니다 — 버튼 설명("KPI 집계 데이터")과 달리 발주 RAW 원본 행을 그대로 내보내던 것을, `computeKPIs()` 실측값 기반의 KPI 13개 지표(영역·지표·목표·선택월 값·달성여부·산식) 요약으로 교체

## 운영·안정성
- html2pdf.js(html2canvas+jsPDF 번들)는 두 PDF 버튼을 **처음 누를 때만** CDN에서 로드됩니다 — 평상시 페이지 로드 속도에는 영향이 없고, 오프라인/차단 환경에서는 버튼 클릭 시 안내 후 중단됩니다
- 사이드바·브라우저 탭 제목의 버전 라벨이 v14로 남아있던 것을 v16으로 정정(v15 배포 시 미갱신분)
- **배포 전 원격 통합**: 이 작업은 로컬이 원격보다 11커밋 뒤처진 상태에서 시작됨(다른 세션의 2025 원본 YoY 업그레이드·이슈현황 탭 필터 수정·W29 CSV 갱신 등) — 원격을 먼저 fast-forward로 받아 그 위에 v16 작업을 얹었고, 코드 충돌은 없었음(변경 영역이 겹치지 않음). 분석 리포트의 2025 로더(`loadPrevYearRaw`)와 원격 KPI YoY의 2025 로더(`load2025RawData`)가 별도로 공존 — 향후 통일 여지 있음
- 실제 CSV(발주 5,924행·이슈 344행 + 2025 RAW 13,529행)로 두 PDF 생성 경로를 브라우저에서 재현해 에러 없이 5개 섹션·차트·표가 모두 렌더링되는지 검증 후 커밋
