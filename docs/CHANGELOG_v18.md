# SCM 대시보드 v18 변경 내역 (2026.07.20)

팀 공유용 — 변경 1건당 1줄 요약입니다. 상세 사용법은 `docs/SCM_DASHBOARD_사용자가이드_v18.html`, 구현 로직은 `docs/SCM_DASHBOARD_로직설명_v18.html` 참고.

v18은 **팀 피드백 5건을 반영한 화면 정리 버전**입니다.

## 이슈 상세 모달 — 협력사 단위 통합
- 이슈 상세 모달의 "제품별·협력사별 이슈 상세"가 **협력사별 이슈 상세**로 바뀌었습니다 — 같은 협력사의 이슈가 제품 구분 없이 한 행으로 합쳐집니다 (이슈건수 내림차순)
- 각 행에 주요 이슈 제품 Top3가 표시되고, 행을 클릭하면 제품·유형·내용·대응이 포함된 개별 이슈 목록이 펼쳐집니다
- 발주건수도 협력사 전체 기준으로 바뀌어 이슈율이 협력사 단위로 계산됩니다

## 재고 분류 기준 — 굿즈코드 STCK 추가
- **굿즈코드에 STCK가 포함되면 전부 재고생산**으로 분류합니다 (기존 movement/project의 '재고' 표기 기준에 추가) — 월 매입금액의 재고제외 집계를 포함해 재고 구분을 쓰는 모든 화면에 일괄 적용

## 재고생산(별도) 카드 제거 — 월 매입금액에 통합
- 종합 현황의 **재고생산(별도) KPI 카드를 없애고**, 재고생산 매입액과 **전체 매입의 몇 %인지를 월 매입금액 카드 비고에 병기**합니다 (KPI 카드 6개 → 5개)
- **매입금액 상세 모달**에 협력사별 **프로젝트 / 재고생산 컬럼**이 추가되어 어느 협력사의 매입이 재고생산분인지 한 표에서 구분됩니다
- **발주 TASK 상세 모달**(발주현황 Top 탭)도 프로젝트 발주 / 재고생산 발주 **두 섹션으로 분리** 표시됩니다 (별도 모달 추가 없음)

## 졸업 검토 — 권장조치 필터
- 졸업 검토 목록 상단에 **권장조치 필터 칩**(전체 / 운영종료 검토 / 졸업 검토 / 모니터링, 건수 표시)이 생겨 조치 유형별로 갈라 볼 수 있습니다

## 발주 현황 — 단일 화면으로 정리
- **발주량 Top 20 / Bottom 20 / 프로젝트별 매입 / 발주 Task 탭을 삭제**하고 **전체 제품 발주량** 하나만 표시합니다 (헤더 클릭 정렬·MoM/YoY·검색 연동은 그대로)
- 발주수량·MoM·YoY·월간발주빈도·연간발주량합·공급가액 숫자 셀에 줄바꿈 금지를 적용해 좁은 화면에서 값이 두 줄로 밀려 보이던 문제를 정리했습니다

## 다음부터
