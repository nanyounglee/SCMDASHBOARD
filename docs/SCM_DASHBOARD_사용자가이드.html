1. 시작하기 — 데이터 업로드

1-1. 접속

브라우저에서 https://nanyounglee.github.io/SCMDASHBOARD/ 접속 또는 로컬 HTML 파일 열기

1-2. CSV 업로드 (2가지 방법)

A
통합 드래그앤드롭 — 상단의 "📁 CSV 파일 드래그 또는 클릭" 영역에 여러 파일을 한번에 드래그. 파일명과 무관하게 헤더 컬럼으로 자동 인식됩니다.
B
개별 슬롯 — "▼ 데이터 업로드" 버튼 클릭 → 상세 영역 펼침 → 원하는 슬롯에 직접 업로드
파츠 마스터 · 2025년 데이터 · 공정DB/단가DB는 영구 임베딩되어 있어 업로드 불필요합니다. 파츠 마스터 변경 시만 "파츠 마스터 갱신" 버튼을 사용하세요.

2. 페이지별 기능 안내 — 외주생산파트

종합 현황

항목확인할 수 있는 것
KPI 카드 5개발주TASK (프로젝트+재고생산+긴급) · 월 매입금액 · 이슈건수 (품질/수량/운영) · 긴급발주 · 고객인지이슈
YoY 모든 카드에 전년 동기 대비 증감률 자동 표시
KPI 클릭 → 모달발주TASK: 전체 굿즈 발주량 내림차순 (미발주 포함)
매입금액: 제조유형별 그룹 → 협력사 매입 내림차순 → 협력사 클릭 시 3개월 추이+Top3 제품
이슈: 품질/수량/운영 탭 필터 + 대응 + 실패비용
긴급: 긴급 발주 목록 (서울디지털인쇄 제외)
차트매입 추이 (월별/주별, 합산/협력사/업태/굿즈코드 전환) · 이슈 유형 도넛 · 담당자별 발주 · 월별 이슈 추이 · 제조유형 매입비중 (업태/카테고리 전환)
검색상단 검색창에 제품명/협력사명/파츠코드 입력 → 모든 KPI·차트가 실시간 필터됨

발주 현황

탭확인할 수 있는 것
전체 제품 발주량선택 월의 모든 산출물 발주수량 내림차순 + MoM 전월대비 + YoY 전년대비 증감률
Bottom 20발주수량 가장 적은 20개 (미발주=0 포함, 빨강 강조)
프로젝트별 매입프로젝트코드별 공급가액 합계 Top 20 (세금계산서 분할 합산)
발주빈도 Top 20산출물별 발주 횟수 내림차순
발주 Task전체/미입하/지연>14일 필터 · 입하경과일 자동 산출 (입하확정일 or 예정일 기준)

매입 현황

항목확인할 수 있는 것
차트협력사별 매입금액 Top 10 (바) · 제조유형 분포 (도넛) · 업태별 매입 (바) · 인쇄유형별 매입 (도넛)
테이블협력사별 매입 상세 Top 20 — 제조유형, 업태, 공급가액, 건수, 비중(%)

이슈 현황

탭확인할 수 있는 것
전체 · 품질 · 수량 · 운영입하일, 프로젝트, 제품, 담당자, 카테고리, 이슈내용, 대응(품질등급의견판단사유_SCM), 실패비용(재제작 취득원가)
실패비용 = 동일 프로젝트에서 task_id에 '재제작'이 포함된 발주의 취득원가 합계. 재제작이 없으면 0원.

고객인지 이슈

항목확인할 수 있는 것
차트연도별 고객인지이슈 추이 · 작성자별 건수
KPI 클릭 → 모달이슈내용 상세 + 실패비용

원가 분석

탭확인할 수 있는 것
월별 원가 요약매출/원가 추이 바차트 · 원가율 추이 라인 · 월별 매출/원가/원가율/마진 테이블
프로젝트별 상세프로젝트 선택 → 매출/투입원가/원가율 KPI → TASK별 실제단가 vs 공정DB 표준단가 비교(%) → Movement 재고투입 → 공정DB 참조
확인된 원가율은 최솟값입니다. 미매칭 자재, 내부 공정, 물류비, 보관비는 미포함.

협력사 현황

항목확인할 수 있는 것
KPI총 협력사 수 · 턴키제작 · 단순구매
테이블협력사명, 업태, 제조유형, 인쇄, 담당MD, 결제조건, 상태 — 검색 가능

협력사유형별 제품군

제조유형(턴키/외주용역/단순구매) × 굿즈코드 히트맵 매트릭스. 셀 색상 강도 = 발주건수 비례. hover 시 매입금액 표시.

하도급법 위험

항목확인할 수 있는 것
KPI총 위험건수 · 최대지연일 · 위험 협력사 수
자동 산출공급망_RAW의 하도급계약 대상여부='대상'인 협력사 + 미입하 + 경과일 ≥ 90일 → 자동 표시

5. 자주 묻는 질문

Q. 파일을 올렸는데 데이터가 안 뜨나요?

상단의 월 칩을 클릭해서 올바른 월이 선택되어 있는지 확인하세요. CSV에 해당 월 데이터가 없으면 KPI가 0으로 표시됩니다.

Q. 2025년 데이터는 따로 안 올려도 되나요?

네. 2025년 데이터는 영구 임베딩되어 있어 YoY 비교가 자동으로 표시됩니다.

Q. 하도급법 위험 CSV를 따로 올려야 하나요?

아니요. 발주_RAW + 공급망_RAW만 올리면 입하경과일과 하도급대상여부를 조합하여 자동 산출됩니다.

Q. 원가분석에서 마진율이 안 나와요

마진율은 공정DB(cost_db.json)의 표준단가를 기반으로 산출됩니다. 공정DB에 해당 굿즈코드가 없으면 '—'으로 표시됩니다.

Q. 시즌계획에서 입력한 확정수량이 사라졌어요

확정수량은 브라우저 localStorage에 저장됩니다. 다른 브라우저나 시크릿 모드에서는 이전 입력이 보이지 않습니다.

Q. 검색하면 KPI 숫자도 바뀌나요?

네. 검색어를 입력하면 모든 KPI, 차트, 테이블이 해당 검색어와 관련된 데이터만으로 필터됩니다. "(필터 적용)" 텍스트가 표시됩니다.
