## 담당자 원칙

| 영역 | 담당 | 요청 대상 업무 |
|---|---|---|
| **구매전략파트 담당 데이터** | **김하은** | S&OP 재고주간·매출월간·재고 KPI 요약 3종 등 GSheets발 CSV의 갱신·오류, 품절리스트 수동 예외(MANUAL_STOCKOUT) |
| **외주생산파트 담당 데이터** | **육승미** | 발주/이슈/협력사/파츠/굿즈마스터 등 Airtable발 데이터의 내용 정합(코스트베이스 단가·파츠 정보 포함), 분기별평가, 협력사 수동 예외(MANUAL_VENDOR_TERMINATIONS) |
| **대시보드 구조·기능·서비스·로직** | **이난영** | 화면/산식/자동화 파이프라인 오류, 신규 기능, 버전 관리·배포, GitHub Actions·토큰 |

데이터 "값"이 이상하면 해당 파트 담당자에게, 화면·계산·자동화가 이상하면 이난영에게 요청합니다.

## 자동 갱신 (사람 개입 불필요 — 실패 시에만 이난영)

| 주기 | 항목 | 방식 |
|---|---|---|
| 매주 목 13:00 KST | 발주(order)·이슈(issue)·협력사(sup)·고객인지이슈(ci)·품절리스트·파츠(parts)·굿즈마스터 등 Airtable CSV 갱신 | GitHub Actions `weekly-airtable.yml` — 이전 버전은 CSV_BANK 주차 폴더에 자동 스냅샷 |
| 매주 목 13:00 KST | 진행현황(progress)·매출결산(project) 주차별 CSV 신규 생성 | 동일 워크플로 |
| 매주 목 13:00 KST | 파츠 단가 변경 감지 → 가격 에폭(data/parts_price_history.json) 추가 | `update_price_history.mjs` — 발주일 시점 단가 계산의 원천 |
| 상시(화면 접속 시) | 파츠 신규/졸업/연결해제·단가 변동·협력사 신규/종료 감지 → 변경 이력(data/change_log.json)·공지 반영 | 대시보드가 스냅샷 체인 비교, GitHub 토큰 보유 브라우저가 자동 커밋 |
| 매년 1/2 00:10 KST | 직전 연도 order/issue를 CSV_BANK/archive/{연도}/로 보존 (YoY 기준) | `yearly-archive.yml` — 1월 첫 주에 폴더 생성 확인만 |
| 수시 | "지금 새로고침" 버튼 — 주간 갱신을 즉시 실행 | 대시보드 상단 (GitHub 토큰 필요, 1~3분+배포 대기 60초) |

## 수동 갱신 (담당자 액션 필요)

| 주기 | 항목 | 방법 | 담당 |
|---|---|---|---|
| 주 1회 | S&OP 재고주간(inv_weekly)·매출월간(sales_monthly)·재고 KPI 요약 3종(dashboard_*) | GSheets에서 CSV Export → 대시보드 업로드 또는 CSV/ 커밋 | **김하은** |
| 주 1회(권장) | 매출결산 총매출(proj_rev) — 원가분석 매출 원천 | Airtable 뷰 CSV export → CSV/proj_rev.csv 교체 (또는 `AIRTABLE_SOURCES` 변수에 등록 시 자동 편입) | **육승미** (자동 등록은 이난영) |
| 분기 1회 | 분기별평가(quarter_eval) — Tier 관계 점수 원천 | GSheets 공급망_분기별평가 export → CSV/quarter_eval.csv | **육승미** |
| 분기 1회 | 시즌재고 계획 CSV | S&OP 시트 export | **김하은** |
| 월 1회(해당 시) | 품절 수동 예외(MANUAL_STOCKOUT)·협력사 거래종료 수동 예외(MANUAL_VENDOR_TERMINATIONS) 상수 갱신 | index.html 상수 수정 요청 | 내용: **김하은/육승미** · 반영: **이난영** |
| 연 1회 + 대규모 단가 개정 시 | 공정DB 임베딩(cost_db.json) 재생성 — 현재 6/24 스냅샷 | 원가 DB 기준 재생성 (단가 값 자체는 주간 parts.csv가 최신 반영 중) | **이난영** |
| 필요 시 | 코스트베이스 파츠 단가·발주서명칭 정비 (미기재 단가 채우기 — 예: 밀크에코컵 승화전사 0원) | Airtable 코스트베이스에서 수정 → 주간 갱신 때 자동 반영·공지 | **육승미** |
| 신규 협력사 등록 시 | 협력사 **업태·인쇄** 컬럼 기입 (Airtable 원천에 없는 수기 데이터 — 기존 값은 주간 갱신에서 자동 보존되므로 신규분만) | CSV/sup.csv 해당 협력사 행에 기입 요청 | 내용: **육승미** · 반영: **이난영** |
