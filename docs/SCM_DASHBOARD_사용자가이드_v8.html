v8 핵심 변경: 구매 재고 화면은 원천 RAW인 inventory_weekly를 직접 계산하기보다, 구글스프레드시트에서 미리 집계한 대시보드용 CSV 3종을 우선 사용합니다.

v8 제외 범위: 매입검토와 시즌재고계획은 기존 구매 대시보드에서도 별도 테이블 기반의 확정 전 기능이므로 이번 통합 범위에서 제외했습니다. 대시보드에는 “추후 반영”으로 표시됩니다.

3. CSV 저장 규칙

  • 구글스프레드시트에서 쉼표로 구분된 값 CSV로 내려받습니다.
  • 파일명은 접두어 없이 dashboard_period_summary.csv, dashboard_group_summary.csv, dashboard_sku_snapshot.csv로 저장합니다.
  • 저장 위치는 저장소의 CSV/ 폴더입니다.
  • CSV/_manifest.json에 등록된 고정 파일명이 자동 로드 기준입니다.
