# SCM 대시보드 v15 변경 내역 (2026.07.15)

팀 공유용 — 변경 1건당 1줄 요약입니다. 상세 사용법은 `docs/SCM_DASHBOARD_사용자가이드_v15.html` 참고.

v15는 두 갈래 작업을 병합한 버전입니다: (1) 난영님이 원격(origin/main)에 직접 올린 월간 공지사항·협력사 거래종료/드릴다운 작업, (2) 같은 시점 별도 세션에서 진행한 기간선택 통합 개편. 두 작업은 서로 모르는 채 갈라졌다가 이번에 `git merge`로 합쳤습니다.

## 기간선택 통합 (신규)
- 외주생산파트 핵심 화면(종합현황·발주·매입·이슈·고객인지이슈)의 기간 필터를 **주별 / 월별 / 분기별 / 기간지정(달력)** 4탭 선택기로 통일 — 이제 임의의 날짜 구간을 직접 지정해서 볼 수 있습니다
- 화면 상단 배지의 MoM/YoY 비교 라벨이 선택한 단위에 맞춰 자동 전환됩니다 (주별→WoW, 월별→MoM, 분기별→QoQ, 기간지정→YoY만 표시 · "전기 대비"가 의미 없는 임의 구간이라 전기 비교는 제공하지 않음)
- 매입금액/매출은 원본 데이터(세금계산서작성월)가 월 단위로만 존재해서, 어떤 단위를 선택해도 항상 월 단위로 집계됩니다 — 화면에 안내 문구로 표시
- 아직 세부 필터 로직이 주/분기/기간지정을 지원하지 않는 13개 화면(발주 진행현황·원가분석·제품별판매추이·협력사·협력사유형별제품군·QMS·하도급법위험·견적계산기·AI어시스턴트·리포트생성·재고현황·포트폴리오·KPI목표)은 동일한 선택기 UI를 보여주되 주별/분기별/기간지정 탭이 회색 비활성 처리되고 "월 단위만 지원 (추후 반영)" 툴팁이 뜹니다 — 세부 구현은 다음 버전에서 순차 반영 예정
- YoY(전년 동기) 비교는 2025년 데이터가 월 단위 사전집계(`data_2025.json`)만 있어서 현재는 월 근사로 계산됩니다 — 2025년 원본(행 단위 RAW) CSV 확보 시 정확한 날짜 기준으로 업그레이드 예정

## 월간 공지사항 — 정확한 날짜 기준으로 수정
- (v14 추가분에서 최초 도입된 기능입니다) 졸업/출시/품절 제품 판정을 "선택 월과 같은 달인지" 비교하던 방식에서, 선택한 기간(SEL_DATE_RANGE)에 **정확한 날짜가 포함되는지**로 변경 — 더 정밀해졌습니다
- 기간지정으로 여러 달(예: 5~7월)에 걸친 구간을 선택하면, 해당하는 모든 달의 졸업/출시/품절/협력사 변동 공지를 합쳐서 보여줍니다 (이전에는 선택 구간의 마지막 달 하나만 기준으로 삼아, 예를 들어 6월 거래종료 협력사를 5~7월 구간에서 못 보는 문제가 있었습니다)
- 신규/거래종료 협력사 스냅샷 비교도 구간에 걸치는 모든 달에 대해 개별 비교한 뒤 합산하도록 확장
