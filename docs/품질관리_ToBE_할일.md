## 2. 내일 해야 할 일 (사용자 액션 — 우선순위순)

### ① Google Form "출하 전 자가검사 성적서" 생성 (30분)
- 문항 9개: `AIRTABLE_품질필드_추가명세.md` §⑤ 표 그대로 (협력사명 **드롭다운** 필수 — sup.csv 표기와 일치, 발주번호 필수, **사진 문항 넣지 말 것** — 넣으면 협력사 Google 로그인 강제)
- 응답 GSheet에서 유입 경로 결정: **ⓐ 파일 → 웹에 게시 → CSV 링크 발급**(권장, 실시간·수작업 0 — 단 링크 공개 상태가 되므로 사내 규정 확인) 또는 ⓑ 주 1회 수동 export
- **링크(또는 export 파일)를 Claude에게 전달** → 대시보드에 검수합격률(전수판)·제출률 지표 연동 구현

### ② CAPA 운영 규칙 결정 (10분 — 결정만 하면 구현은 Claude가)
- **대상 범위**: 모든 이슈에 CAPA를 달지, `품질이슈대응필요여부`=필요 건만 달지 (권장: 필요 건만 — 경미한 운영/수량 이슈까지 강제하면 입력 부담)
- **원인분류 옵션 확정**: 초안 `원자재/인쇄·컬러/공정·가공/포장·구성/설계·사양/물류·취급/검수누락/기타` — 팀 용어로 수정 가능, 단 **한번 정하면 불변** (이력 일관성)
- 결정되면 → 이슈 목록 인라인 입력 UI(상태·기한·원인분류) + `data/capa_overrides.json` + 기한 초과 하이라이트 + CAPA 트래커 탭 구현

### ③ 협력사 분기 성적표(scorecard) 지표 확정 (10분)
- 기존 분석 리포트 PDF에 이미 있는 것: 연간요약·YoY·추세·Top5·재발이슈
- **추가 후보 (선택)**: 이슈율·유출 건수·실패비용(재제작)·분기평가 등급(A/B/C/D)·Tier·검수합격률(IQC 가동 후)
- 어떤 지표를 담아 협력사에 송부할지 확정되면 리포트 확장 구현

### ④ (선택) Airtable 토큰에 `schema.bases:read` 스코프 추가 (5분)
- airtable.com/create/tokens → 기존 토큰 편집 → 스코프 추가 (재발급·시크릿 교체 불필요)
- 효과: `airtable-discover` §A에서 전체 Base/Table/View 목록 열람 가능 (필수는 아님 — 프로브가 핵심 질문에 이미 답함)

### ⑤ (보류 중 확인) 월간 리포트 관련 2건
- **Tier3/4 모수 정합**: 리포트에서 제외하는 협력사 명단(활성 미확인 4개사·해외 단순구매 미거래 6개사) 확보 → 전달 시 즉시 반영
- **Tier 산식의 이슈 조인 전환**: 현재 Tier 점수는 리포트 정합을 위해 구(접두어) 조인 유지 중 — 리포트 담당과 "정확 조인으로 전환" 합의 여부 결정
